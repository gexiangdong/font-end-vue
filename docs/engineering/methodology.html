几个不同的实现方法

会哪些技术的人是我的同行？

以工程化的方式进行前端开发，目前比较有名气的框架有三个：

  • Angular
  • React
  • Vue.js

Angular

第一版发布于2010年，叫做AngularJS，自第二版改名为Angular，Google维护的开源JavaScript库。

React

第一版发布于2013年，Facebook开发的，开源JavaScript库.

Vue.js

第一版发布于2014年，作者尤雨溪。
